This is a simple BuyMeACoffee Dapp based on [Next.js](https://nextjs.org/)

## Getting Started

First, run the development server:

```bash
npm install
```

or

```bash
yarn install
```

Then, run:

```bash
npm run dev
```

or

```bash
yarn run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
